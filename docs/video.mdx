# Video Compliance

ZebraTruth supports native video compliance checks through `POST /v1/compliance/check-video`.

No client-side ffmpeg is required. Upload the video with `POST /v1/compliance/media/upload-url`, submit the returned `blobPath`, then poll `GET /v1/compliance/check-video/{requestId}` for timestamped findings.
